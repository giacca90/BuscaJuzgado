# BuscaJuzgado

## FrontAngular

Este es el Front de la app BuscaJuzgado. Está escrito con Angular 17 Standalone.

## Características

> Tema Oscuro o Tema Claro Automático
> 
> Angular Material SOLO para los Modales
> 
> Responsive con CSS, FlexBox y MediaQuery
